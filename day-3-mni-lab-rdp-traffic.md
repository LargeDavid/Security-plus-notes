# Day 3 Mini Lab – Suspicious RDP Traffic

## Scenario
Unusual traffic is detected on port 3389.

## What is Port 3389?
Used for Remote Desktop Protocol (RDP)

## Possible Risks
- Unauthorized remote access
- Brute force login attempts
- Compromised credentials

## What Should Be Done
- Restrict RDP access
- Enable MFA
- Monitor login attempts
- Change default port if necessary

## SOC Analyst Thinking
- Check login logs
- Identify source IP addresses
- Look for repeated failed logins

## Key Takeaway
Open remote access ports are common attack targets.
