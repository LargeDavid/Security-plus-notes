# IDS vs IPS

## IDS (Intrusion Detection System)
- Monitors traffic
- Detects suspicious activity
- Sends alerts
- Does NOT block attacks

## IPS (Intrusion Prevention System)
- Monitors traffic
- Detects and blocks attacks
- Works in real time

## Types

### HIDS
Installed on a host system

### NIDS
Monitors network traffic

## Detection Methods

- Signature-based: detects known threats
- Anomaly-based: detects unusual behavior

## Key Takeaway
IDS alerts. IPS acts.
