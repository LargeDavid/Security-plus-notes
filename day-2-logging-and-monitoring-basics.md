# Logging and Monitoring Basics

## What is Logging?
Logs are records of system and user activity.

## Types of Logs
- System logs
- Firewall logs
- Application logs
- IDS/IPS logs

## Why Logging Matters
- Detect suspicious activity
- Investigate incidents
- Track user behavior

## SOC Perspective
Security analysts rely on logs to identify threats and respond to incidents.

## Key Takeaway
Without logs, there is no visibility into what is happening in a system.
