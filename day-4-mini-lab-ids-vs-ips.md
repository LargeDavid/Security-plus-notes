# Day 4 Mini Lab – IDS vs IPS Scenario

## Scenario
A company uses an IDS but experiences a successful attack.

## Why This Happened
IDS only detects and alerts — it does not block attacks.

## What Could Have Prevented It
Using an IPS to actively block malicious traffic.

## Analyst Thinking
- Review IDS alerts
- Identify attack pattern
- Recommend IPS implementation
- Improve monitoring rules

## Key Takeaway
Detection alone is not enough — prevention is critical
