# Common Cyber Attacks

## Social Engineering
- Phishing: fake emails
- Spear phishing: targeted attack
- Vishing: voice-based attack
- Smishing: SMS-based attack

## Malware
- Virus: attaches to files
- Worm: spreads automatically
- Trojan: disguised as legitimate
- Ransomware: locks data for payment
- Spyware: monitors activity

## Password Attacks
- Brute force: tries all combinations
- Dictionary attack: uses wordlists
- Credential stuffing: reused credentials

## Key Takeaway
Many attacks succeed by exploiting users, weak passwords, or unpatched systems.
