# Security Controls 

## By Function

### Preventive Controls
Stop attacks before they happen  
Example: Firewalls, access control systems

### Detective Controls
Identify attacks in progress or after  
Example: Logs, IDS

### Corrective Controls
Fix issues after an attack  
Example: Patching, restoring backups

### Deterrent Controls
Discourage attackers  
Example: Warning signs, policies

### Compensating Controls
Alternative controls when primary ones are not possible

### Directive Controls
Guide behavior  
Example: Security policies

---

## By Category

### Technical Controls
Implemented through systems  
Example: Firewalls, encryption

### Managerial Controls
Policies and planning  
Example: Risk assessments

### Operational Controls
Day-to-day procedures  
Example: Security training

### Physical Controls
Protect physical assets  
Example: Locks, CCTV



## Key Takeaway
Security controls are layered to prevent, detect, and respond to threats effectively.
