# Networking Basics

## OSI Model (Simple View)
Network communication happens in layers, each responsible for a specific function.

## Common Protocols

- HTTP (80): Web traffic
- HTTPS (443): Secure web traffic
- SSH (22): Secure remote login
- RDP (3389): Remote desktop access
- DNS (53): Resolves domain names to IP addresses
- DHCP (67/68): Assigns IP addresses

## Network Devices

- Router: Connects networks
- Switch: Connects devices in a network
- Firewall: Filters traffic
- Proxy: Acts as intermediary between user and internet

## Security Concepts

- Segmentation: Dividing networks to limit attacks
- Zero Trust: Never trust, always verify
- NAT: Hides internal IP addresses

## Key Takeaway
Understanding how networks work helps identify where attacks happen and how to defend against them.
